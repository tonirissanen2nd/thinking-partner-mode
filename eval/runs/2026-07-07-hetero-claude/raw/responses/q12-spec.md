Two caveats worth stating, since "sea level" is doing real work in the question:

- The 100°C figure is fixed at standard atmospheric pressure (1 atm / 101.325 kPa). "Sea level" is a good proxy for that but not identical — actual sea-level pressure varies with weather (roughly 95–105 kPa), which moves the boiling point by about ±1°C on ordinary days.
- Since 2019 the Celsius scale is defined via the kelvin and the Boltzmann constant, so 100°C is no longer *exactly* water's boiling point by definition — the measured value at 1 atm is about 99.97°C. The 0.03°C gap is immaterial for any practical purpose.
